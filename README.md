# Shopnado

Shopnado is a modern, full-stack e-commerce application designed to deliver a seamless shopping experience with a clean, responsive interface and robust backend architecture.  
It offers all the essential features of a real-world online store â€” from dynamic product listings to secure checkout and admin management tools.

---

## Features

- **Modern Interface:** Responsive, intuitive design built for all devices.  
- **Dynamic Catalog:** Real-time product rendering and filtering.  
- **Secure Authentication:** User login, registration, and protected routes using JWT.  
- **Shopping Cart & Checkout:** Full cart functionality with order tracking.  
- **Payment Integration:** Stripe API for secure, fast payments.  
- **Admin Dashboard:** Product, user, and order management with analytics overview.  
- **Cloud Ready:** Easy to deploy on platforms like Vercel, Render, or AWS.  

---

## Tech Stack

| Layer | Technologies |
| :---- | :------------ |
| **Frontend** | React, Vite, TailwindCSS, Redux Toolkit |
| **Backend** | Node.js, Express.js |
| **Database** | MongoDB with Mongoose |
| **Authentication** | JWT, bcrypt, OAuth (Google) |
| **Payments** | Stripe API |
| **Deployment** | Vercel / Render / MongoDB Atlas |
